Overview

The video covers the future of dataframes and data systems, featuring Wes McKinney, the creator of pandas and Apache Arrow. Several key software projects are discussed, focusing on data interchange, analysis, and storage.

Key Software Projects

  • Pandas: An open-source library providing data structures and data analysis tools for Python.
  • Apache Arrow: An open-source project for in-memory data interchange.
  • Ibis: An open-source project aiming to bridge the world of relational databases and data science tools.
  • Spark: An open-source unified analytics engine for large-scale data processing.
  • Presto: An open-source distributed SQL query engine.
  • SQL: A structured query language.
  • Iceberg: An open-source table format for big data.
  • Parquet: An open-source columnar data file format.

Key Points from the Video

Apache Arrow is highlighted as a language-agnostic data interchange standard being used in composable data systems. Ibis is noted for allowing SQL queries to be run on various data sources.

One challenge of composable data systems is the absence of a unified data frame interface, but multiple projects are working towards standardizing data frames. Another challenge is managing unstructured data within these systems, with approaches like storing unstructured data separately from structured data being explored.
